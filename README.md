# 🌍 World Explorer

A high-performance, 3D interactive globe for exploring the world from your browser. This project is a frontend-only tool designed for fun and discovery.

## ✨ Features

- **3D Globe Engine**: Powered by CesiumJS for high-fidelity satellite and terrain rendering.
- **Street Explorer**: **Mapillary** (global; your client token stays out of git) with a **Panoramax** fallback (open data, no key) where Mapillary is unavailable or has no coverage.
- **Random Discovery**: Jump to random land locations or curated interesting places.
- **Cinematic Mode**: Experience the world with smooth, automated camera movements.
- **Location Data**: Real-time geocoding, weather, and country information.
- **Personalized**: Save your favorite spots and track your exploration history.
